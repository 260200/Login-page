JavaScript

JS is a lightweight, interpreted, or just-in-time compiled programming language with first-class functions. While it is most well-known as the scripting language for Web pages, many non-browser environments also use it, such as Node.js, Apache CouchDB and Adobe Acrobat.

JavaScript is a client scripting language which is used for creating web pages. It is a standalone language developed in Netscape. It is used when a webpage is to be made dynamic and add special effects on pages like rollover, roll out and many types of graphics.

The biggest advantage of JavaScript is it's ability to produce the same result on all modern browsers. Client-Side execution: No matter where you host JavaScript, execute always on client environment to save a bandwidth and make execution process fast.
